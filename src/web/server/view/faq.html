よく頂く質問

理科大ってどこにあるの？
東京理科大学には下記の5つのキャンパスがあります。機会がありましたら是非、足をお運びください。 メンバーは全員、野田キャンパスに所属しています。
  • 神楽坂キャンパス神楽坂校舎
  • 神楽坂キャンパス富士見校舎
  • 葛飾キャンパス
  • 野田キャンパス
  • 北海道・長万部キャンパス
プライバシーが保護されているか心配です
当サイトでは、画像をAIで解析し人数をカウントしておりますが、外部とのデータのやり取りはトイレの利用を待っている人数のみに限定しております。
そのため、個人情報の漏洩の心配はありません。安心してご利用頂けます。
詳しくは個人情報の取り扱いをご覧ください。

詳しくは利用方法をご覧ください。
トイレの場所がわかりません
当サイトでは、トイレの場所をご案内しております。

女子トイレの混雑状況は確認できないのですか？
開発メンバーが全員男性につき、現在、女子トイレの混雑状況は確認できません。
申し訳ございません。ご理解いただけますと幸いです。
トイレの混雑状況はどのようにして確認しているのですか？
当サイトでは、トイレの混雑状況を撮影した画像をAIで解析し、人数カウントや待ち時間の推定を行っております。
詳しくは利用方法をご覧ください。
トイレの混雑状況はどのくらいの頻度で更新されていますか？
当サイトでは、AIが解析した人数を、トイレの設置場所に応じた通信回線でクラウドにアップロードしております。
通信回線の利用率によっては若干の異なる場合がありますが、概ね以下の通りです。
  • 東京理科大学7号館  (WiFiで通信)    ：30秒/回
  • 東京理科大学16号館(ELTRESで通信)：1分/回
