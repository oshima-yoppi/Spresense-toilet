利用方法

エリアの選択

  • 画面上部、または下部のトイレ一覧から該当するエリアを選択してください。

混雑状況の確認

  • ページ遷移後、表示されているトイレの混雑状況および想定待ち時間を確認してください。
    　混雑状況は、非常に空いている、空いている、混雑している、非常に混雑しているの4段階で表示されます。
    　想定待ち時間は目安です。利用までの待ち時間を保証するものではありません。

最適なトイレの選択と利用

  • 混雑状況と待ち時間から、空いているトイレをご自身で選択いただき、 マップの案内に従ってご利用ください。
